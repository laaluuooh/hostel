# Smart Hostel

Project structure:

- `frontend/` -> React + Vite frontend
- `backend/` -> Node.js backend

## Run backend

```bash
cd backend
npm start
```

Backend runs on `http://localhost:5000`.

## Run frontend

```bash
cd frontend
npm run dev
```

## Optional frontend env

Create `frontend/.env`:

```bash
VITE_API_URL=http://localhost:5000
```
